# VibeCraft Manifest

**VibeCraft — это методология DIY-инжиниринга эмоций для цифровых продуктов.**

Этот репозиторий содержит официальный манифест и стартовые артефакты, фиксирующие ключевые концепции VibeCraft. Его цель — закрепить приоритет идеи и предоставить основу для её дальнейшего развития и внедрения.

### Что здесь находится?

*   **`MANIFEST.md`**: Основной документ, описывающий философию, принципы и ключевые концепции VibeCraft.
*   **`vibe.config.json`**: Пример машиночитаемой декларации «вайба» приложения.
*   **`theme.ts`**: Пример файла темы для фронтенд-проекта, основанный на `vibe.config.json`.
*   **`LICENSE.md`**: Лицензия, по которой распространяются материалы (текст — CC BY 4.0, код — MIT).
*   **`/docs`**: Папка для сгенерированных документов, включая официальный PDF-манифест.

### Позиционирование

VibeCraft сознательно отделяется от термина «vibe coding». Если «vibe coding» — это про ускорение разработки кода с помощью ИИ, то **VibeCraft — это про архитектурное проектирование ощущений**, где ИИ выступает инструментом ремесленника (VibeCrafter), а не просто генератором кода.

### Как начать?

1.  Прочтите **`MANIFEST.md`**.
2.  Изучите артефакты (`vibe.config.json`, `theme.ts`).
3.  Используйте эту методологию в своем следующем проекте, чтобы создать продукт с уникальным характером.
